# GHT (**G**it**H**ub **T**UI)
A fast and simple TUI (Text User Interface) for GitHub.

## Installation
```bash
$ git clone https://github.com/mazylol/ght.git
$ cd ght
$ cargo isntall --path .
```

## Usage
```bash
$ ght
```
